Taller: Probabilidad aplicada a movilidad urbana

Análisis de tiempos de desplazamiento de estudiantes hacia la universidad

Descripción del proyecto

Este proyecto implementa una aplicación en Streamlit para analizar los tiempos de desplazamiento de los estudiantes hacia la universidad, calculando frecuencias, probabilidades empíricas y probabilidades acumuladas, y visualizando los resultados mediante histogramas.

Los datos provienen de un formulario donde cada estudiante reporta su localidad, medio de transporte y tiempos de viaje en horas pico y horas valle.

Cómo ejecutar localmente

1. Clonar este repositorio:

git clone https://github.com/TU_USUARIO/taller-movilidad-streamlit.git
cd taller-movilidad-streamlit

2. Crear un entorno virtual (opcional pero recomendado) e instalar dependencias:

pip install -r requirements.txt

3. Ejecutar la aplicación de Streamlit:

streamlit run app.py

4. Abrir el navegador en la URL que indica Streamlit (normalmente http://localhost:8501).
